# YouTube Streaming Best Practices

## Pre-Stream Checklist
- [ ] Test audio/video sources (OBS, camera, mic)
- [ ] Verify stream key and encoder settings
- [ ] Check internet upload speed (min 5 Mbps for 1080p)
- [ ] Set up stream title, description, and tags

## Encoder Settings (OBS)
- **Video**: 1080p @ 30fps, bitrate 4500-6000 Kbps
- **Audio**: 160-192 Kbps, sample rate 48kHz
- **Keyframe interval**: 2 seconds
- **Preset**: Hardware encoding (NVENC/AMF) for low CPU usage

## Engagement Tips
- Use a countdown screen before going live
- Greet viewers by name in chat
- Keep streams under 2 hours for higher retention
- Add end screen elements for subscribe/like

## Common Pitfalls
- Forgetting to disable copyrighted music
- Low bitrate causing pixelation
- Not monitoring chat for technical issues

> *"Consistency and quality beat perfection every time."*
